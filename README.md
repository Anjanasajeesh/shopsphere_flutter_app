# ShopSphere Flutter App

Flutter e-commerce UI with API integration.

## Features
- Login screen
- Product list API
- Product detail API
- User profile API
- Bottom navigation
- Bloc Cubit state management

## APIs Used
https://api.escuelajs.co/api/v1/products  
https://api.escuelajs.co/api/v1/products/{id}  
https://api.escuelajs.co/api/v1/users/1

## Architecture
Clean Architecture  
Bloc Cubit State Management
